What Is KenKen?

KenKen is a number puzzle that combines Sudoku-style logic with arithmetic. You fill an N x N grid with the digits 1 to N, following two rules: no repeats in any row or column, and groups of cells ("cages") must produce a target number using a given operation.

Reading the Board

The grid is divided into cages — groups of cells outlined with thicker borders. Each cage has a label in the top-left cell showing a target number and an operation (e.g., "12x", "3+", "2-", "2/").

A solved 4x4 KenKen. Each cage satisfies its arithmetic target.

"6x" cage (top-left): contains 3 and 2. 3 x 2 = 6. Correct.

"2/" cage: contains 4 and 2. 4 / 2 = 2. Correct.

"8+" cage (top-right): contains 1, 3, 4. 1 + 3 + 4 = 8. Correct.

The Rules

1. Fill every cell with a digit from 1 to N (where N is the grid size — 4, 6, or 7 depending on difficulty).

2. Each row and each column must contain every digit exactly once (like Sudoku).

3. The digits in each cage must produce the target number using the given operation (+, -, x, /).

4. For subtraction and division, the order matters: the larger number comes first. A "2-" cage with cells 5 and 3 means 5 - 3 = 2.

Strategy Tips

Start with single-cell cages — they just tell you the answer directly. Then look for cages with only one possible combination (e.g., "1-" in a 4x4 can only be {1,2}, {2,3}, or {3,4}).

Use process of elimination: if a row already has digits 1, 2, 3 filled, the remaining cell must be 4.

Controls

Click a cell to select it, then use the numpad or keyboard 1-9 to place a digit. Press 0 or Delete to erase.

Notes mode — toggle with the Notes button or N. In notes mode, tapping a number adds/removes a pencil mark.

Undo — reverses your last action. Hint — reveals the correct digit for the selected cell (-500 pts).

When every cell is filled, press Submit to check your solution.
