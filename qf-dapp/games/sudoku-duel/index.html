Fill the 9x9 grid so every row, column, and 3x3 box contains the digits 1-9 exactly once.

  • Given numbers (bright) are fixed — fill the empty cells
  • Each placement is validated by the server — no guessing
  • 3 mistakes and it's game over
  • Up to 5 hints available (each costs 200 points)
  • Use Notes mode to pencil in candidates

Scoring: 5,000 base points minus penalties for mistakes (-300) and hints (-200), multiplied by a time factor. Faster clean solves score highest.
