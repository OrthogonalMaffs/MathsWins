Number Logic

Kakuro

Crossword meets Sudoku. Fill white cells with digits 1-9 so each run sums to its clue. No repeats in a run.

Fill white cells with digits 1-9 so each horizontal and vertical run sums to its clue number. No digit repeats within a run.

  • Dark cells with numbers are clues — the top-right number is for the run going right, bottom-left is for the run going down
  • Each run is a consecutive group of white cells
  • Digits 1-9 only — no zeros, no repeats within a single run
  • The same digit CAN appear in the same row or column if it's in a different run

Start with short runs: a 2-cell run summing to 3 must be {1,2}, a 2-cell run summing to 17 must be {8,9}. These forced combinations unlock the rest.

How to Play Kakuro

Example: A 2-cell run with clue 7
Possible: 1+6, 2+5, 3+4 (any order)
Not allowed: 3+3 (no repeats)
