Number Logic

Kakuro

Crossword meets Sudoku. Fill white cells with digits 1-9 so each run sums to its clue. No repeats in a run.

Goal: fill every white cell with a digit 1–9 so each run of consecutive white cells sums to its clue. No digit may repeat within the same run.

Reading a clue cell. Dark cells with split numbers are clues. The gold number (bottom-right) is the target sum for the run going right. The silver number (top-left) is the target sum for the run going down. A clue may have one or both.

Run rules. A run is the group of white cells immediately to the right of (or below) a clue, stopping at the next dark cell or grid edge. Digits 1–9 only; within a single run every digit is different. The same digit can reappear in the same row or column if it belongs to a different run.

The unlock trick — forced combinations. Short runs at extreme sums have only one possible set of digits. Find these first, then cross-check where runs intersect.

  • 2 cells summing to 3 → {1, 2}
  • 2 cells summing to 4 → {1, 3}
  • 2 cells summing to 16 → {7, 9}
  • 2 cells summing to 17 → {8, 9}
  • 3 cells summing to 6 → {1, 2, 3}
  • 3 cells summing to 7 → {1, 2, 4}
  • 3 cells summing to 23 → {6, 8, 9}
  • 3 cells summing to 24 → {7, 8, 9}

Where a forced run crosses another, the intersection pins specific digits. That is how the grid unravels.

Controls. Tap a white cell and enter a digit with the numpad. Notes lets you pencil in candidates (no score cost). Sums opens the combination helper for the currently selected run (−10% final score). Hint reveals a single cell (−500). Submit checks the whole grid — you have three attempts before a game-over pity score.

How to Play Kakuro

Goal. Fill every white cell with a digit 1–9 so each run of consecutive white cells sums to its clue. No digit repeats within the same run.

Reading a clue cell. The gold number (bottom-right) is the sum of the run going right. The silver number (top-left) is the sum of the run going down. A clue may have one or both.

Runs. A run is the white cells immediately to the right of (or below) a clue, stopping at the next dark cell or grid edge. Within one run every digit must be different. The same digit may reappear in the same row or column if it’s in a different run.

Example — 2-cell run summing to 7:
Possible: {1,6}, {2,5}, {3,4} in either order.
Not allowed: {3,3} — no repeats.

Unlock trick — forced combinations. At extreme sums the digit set is unique. Solve these first, then use intersections.

  • 2 cells = 3 → {1, 2}   |   4 → {1, 3}   |   16 → {7, 9}   |   17 → {8, 9}
  • 3 cells = 6 → {1, 2, 3}   |   7 → {1, 2, 4}   |   23 → {6, 8, 9}   |   24 → {7, 8, 9}
  • 4 cells = 10 → {1, 2, 3, 4}   |   30 → {6, 7, 8, 9}

Where a forced run crosses another, the intersecting cell must use a digit that exists in both sets. That constraint is how Kakuro unravels.

Controls. Tap a cell, enter a digit from the numpad. Notes pencils candidates (no cost). Sums opens the combination helper for the current run (−10% final score). Hint reveals one cell (−500). Submit checks the whole grid — three attempts before game-over pity score.

Tip. Start with the shortest runs and the most extreme sums — they commit you to specific digits. Use Notes liberally in longer runs until intersections narrow things down.
