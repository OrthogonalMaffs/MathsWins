Arithmetic Logic

KenKen

Fill the grid with 1-N. No repeats in rows or columns. Cages must hit their arithmetic target.

Fill the grid with digits so no number repeats in any row or column, and each cage's numbers combine to its target using the given operation.

  • A 4x4 grid uses digits 1-4, a 6x6 uses 1-6, a 7x7 uses 1-7
  • Cages show a target number and an operation (+, -, x, /)
  • Single-cell cages are freebies — the target IS the answer
  • For subtraction and division, order doesn't matter

Start with single-cell cages and small cages with few possibilities. Use row/column constraints to eliminate candidates.
